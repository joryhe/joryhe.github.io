发表于 2016-05-30   |   分类于 翻墙   |     |     |   阅读次数   |   字数统计: 952(字)   |   阅读时长: 4(分)

摘要

Shadowsocks（中文名称：影梭）是使用Python等语言开发的、基于Apache许可证开源的代理
软件。Shadowsocks使用socks5代理，用于保护网络流量。在中国大陆被广泛用于突破防火长城（GFW）
，以浏览被封锁的内容。Shadowsocks分为服务器端和客户端。在使用之前，需要先将服务器端部署
在支持Python的服务器上面，然后通过客户端连接并创建本地代理。
目前开发者已宣布停止维护。但仍有更新陆续推送.

原作者：Clowwindy
开发者：Shadowsocks贡献者
操作系统：跨平台（基于客户端与远程）服务器
类型：本地socks5代理
